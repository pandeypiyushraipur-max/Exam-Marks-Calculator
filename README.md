# Exam Score Calculator

This is a simple Java-based calculator that helps students determine their final exam scores based on correct and incorrect answers.

### Formula Used:
`Total Score = (Correct * Weight) - (Incorrect * Penalty)`

### How to Run:
1. Download `ScoreCalculator.java`.
2. Open your terminal.
3. Compile: `javac ScoreCalculator.java`
4. Run: `java ScoreCalculator`
